# Awesome Books
In this exercise I  build a simple yet powerful webpack boilerplate, which I can later use as a starting point in my projects. 

 ## Learning objectives
- Use webpack to bundle JavaScript.

## Built With

- HTML, CSS, JavaScript
- lodash
- npm
- html-webpack-plugin
- webpack-dev-server


## Getting Started
Open index.html

### To launch the project locally:

#### Step 1:
- Clone this repository in your local machine terminal:

git clone <code>https://github.com/Okechukwu-muokwugwo/Webpack-setup.git</code>

#### Step 2:

- Open the html pages in a web browser preferrably Chrome.
